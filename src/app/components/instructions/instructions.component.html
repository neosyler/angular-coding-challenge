  1. Make an HTTP GET request to obtain the listing of products from https://dummyjson.com/products
  2. Display the list of products from that API and include the image (thumbnail), title, description and price.
  3. Add an input to filter the list of products based on the search criteria. This should be case-insensitive and a match is found if the text entered is contained within the name of the product.
  4. Add a "Buy" button to each product. Clicking the button should increment the # of products in the cart on the 'Cart (0)' button.
  5. Create a new page, and a route, to display the contents of the cart. The page can be accessed by clicking on the cart button.
  6. Include a subtotal on the cart page that would display the total cost of all products in the cart.
